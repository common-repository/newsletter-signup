 == Description ==
 
Simple plugin to add a newsletter sign-up form to your website
 
 == Installation ==

1. Download 'newsletter-signup'.
2. Upload the 'newsletter-signup' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'newsletter-signup' from your Plugins page.
4. Initiate and use the plugin in your code by using the shortcode ['newsletter']
5. Have fun! Send an email to info@hemmes.it or visit http://www.hemmes.it/wordpress-plugin-development/newsletter-signup if you have any questions related to this project.
